超图 vs 传统图

突破传统图神经网络局限，实现更精准的分子表示

传统图表示

  • 二元关系：每条边只能连接两个节点
  • 局部信息：难以捕获环结构、官能团等高阶相互作用
  • 信息损失：忽略了分子中的多体相互作用和长程依赖

超图表示

  • 多元关系：超边可连接任意数量节点
  • 高阶信息：自然捕获环结构、官能团、氢键等高阶相互作用
  • 完整表示：保留分子中的多体相互作用和结构层次信息
